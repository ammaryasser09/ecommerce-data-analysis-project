# 💎 Ecommerce Data Analysis Dashboard

A modern interactive **Streamlit dashboard** for analyzing ecommerce data, exploring customer behavior, product performance, and business insights using Python, Pandas, and Plotly.

---

## 🚀 Project Overview

This project provides an end-to-end data analysis solution including:

- 📊 Exploratory Data Analysis (EDA)
- 👥 Customer Segmentation
- 🛒 Product Performance Analysis
- 📈 Revenue & Trend Analysis
- 🔍 Multivariate Relationships
- 📉 Univariate Analysis

---

## 🧠 Key Features

### 🏠 Home Page
- Dataset preview
- Data dictionary (schema explanation)
- Clean UI overview

### 📊 Dashboard
- KPIs (Revenue, Customers, Orders)
- Gender analysis
- City performance
- Category insights

### 📈 Univariate Analysis
- Single variable distribution
- Histograms & bar charts
- Statistical summaries

### 🔥 Multivariate Analysis
- Correlation analysis
- Scatter plots between variables
- Relationship insights between features

---

## 🛠️ Tech Stack

- Python 🐍
